# Capstone 3 – Power BI Sales & Shipping Analysis

## Project Overview

This project was completed using Power BI to analyze bookstore sales and shipping data. The report focuses on East Region sales performance (New York and Connecticut) while also providing company-wide shipping insights.

## Features

- Sales Overview Dashboard
- Book Sales Analysis
- East Region Business Insights
- East Region Shipping Cost Analysis
- Company-Wide Shipping Analysis
- Interactive Drillthrough Page

## Key Findings

- New York generated higher sales than Connecticut.
- Technology and educational products were the strongest performing categories.
- Final shipping costs were consistently lower than estimated shipping costs.
- California generated the highest shipping costs nationwide.

## Tools Used

- Power BI
- Power Query
- DAX
- GitHub
